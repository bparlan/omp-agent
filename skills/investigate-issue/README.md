# Investigate Issue

## Overview

Investigates implementation issues and produces technical understanding.

## Documentation

- **[SKILL.md](./SKILL.md)** — Detailed skill definition and instructions
- **[INDEX.md](../../INDEX.md)** — Complete skill catalog

## Purpose

Understand why implementation doesn't work and recommend fixes.

## Key Responsibilities

- Read implementation artifacts (code, tests, spec)
- Run tests to reproduce issue
- Analyze stack traces and errors
- Identify root causes
- Document findings and recommended fixes

## Out of Scope

Applying fixes, running hotfix workflows, archiving issues
